FUTURE_DS_01 — Retail Sales Analysis Dashboard (Power BI)

Project scaffold for an end-to-end retail sales analysis and visualization workflow.
Built to clean and analyze historical retail data, generate category-wise insights, and present results through an interactive Power BI dashboard for business decision-making.
Project Overview

This project presents an interactive Retail Sales Analysis Dashboard developed using Power BI.
The dashboard analyzes sales performance across multiple product categories and states, focusing on:

Profit distribution

Sales targets

Quantity sold

Category-wise contributions

The project demonstrates the effective use of Business Intelligence techniques to convert raw retail data into actionable insights.



Objectives:

Analyze profit distribution across product categories

Compare sales targets by category

Visualize quantity sold across different product segments

Evaluate state-wise profit contribution

Provide a clear category-level performance overview



Dataset

Source: Kaggle – Retail Sales Dataset
Type: Transactional retail sales data



Key Fields Used:

Category (Clothing, Electronics, Furniture)

Sales Amount

Quantity

Profit

State

Target Sales



Tools & Technologies:

Power BI Desktop

DAX (Data Analysis Expressions)

CSV Dataset (Kaggle)

Power Query for Data Cleaning


Dashboard Highlights:

KPI cards for category-wise sales performance

Profit distribution analysis by category

Quantity contribution using Tree Map visualization

State-wise profit analysis using Donut Chart

Interactive slicers for category-level filtering
